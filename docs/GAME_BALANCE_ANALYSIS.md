### P2级增强（低优先度）

#### 缺失机制补充

**LASER激光炮射速升级**
- 问题: 基础120ms，Lv10仍为120ms，无升级效果
- 方案: 每级降低5ms
  ```
  Lv1: 120ms  →  Lv10: 75ms (目标密度↑60%)
  ```

**PLASMA等离子炮升级幅度加大**
- 问题: 600ms→400ms升级空间小
- 方案: 改为600ms→350ms
  ```
  升级幅度: 25ms/级
  Lv1: 600ms  →  Lv10: 375ms
  ```

**灼烧系统实现**
- MAGMA武器触发灼烧效果
- 敌人受灼烧: 每秒1-2点伤害，持续3秒
- 自身受灼烧: 减速30%

**反弹机制完善**
- SHURIKEN飞镖最多反弹3次
- 反弹间隔: 150ms
- 反弹损伤: 每次反弹伤害-20%
