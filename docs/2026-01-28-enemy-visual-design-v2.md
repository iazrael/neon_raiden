## 设计原则与框架

### 核心定位

通过视觉语言快速传达敌人的行为模式和威胁等级，同时保持简约几何的美学风格。

### 艺术风格：简约霓虹几何

- **基础**：三角形、矩形、圆形等基础几何的组合
- **轮廓**：必须具备战机的可识别特征（机身、机翼、推进器）
- **光效**：高对比度霓虹发光，边缘清晰，无渐变
- **去除**：无装甲板、铆钉、纹理等写实细节

### 颜色-稀有度映射（热色梯度）

```
Common (普通)     → 青色/蓝色  (#00FFFF, #0080FF)
Uncommon (少见)   → 黄色/橙色  (#FFFF00, #FF8000)
Rare (稀有)       → 红色/紫色  (#FF0000, #FF00FF)
Epic (史诗)       → 金色/白色  (#FFD700, #FFFFFF)
```

### 形状-行为映射（战机轮廓类型）

```
直线向下   → 流线型箭头战机（尖锐，指向下方）
追踪冲撞   → 猎鹰形战机（突出前部传感器）
正弦波     → 三角翼战机（宽翼，对称设计）
侧向移动   → 横宽型战机（左右延伸）
圆形/特殊  → 多轴对称平台（无明确前方）
```

### 通用构成元素

所有战机都必须包含以下基本元素：

1. **机身** - 用三角形/菱形表示主体
2. **机翼** - 用三角形/梯形表示
3. **推进器** - 用小三角形/矩形表示，带发光拖尾
4. **能量核心** - 用圆形/菱形表示，位于机身中央

#### 3. 文件大小优化

- 复用几何元素 (`<defs>` + `<use>`)
- 使用简单的路径而非复杂形状
- 动画使用 CSS/SVG 原生特性
- 避免滤镜嵌套过深

#### 4. 碰撞盒对齐

- 所有 SVG 的中心点在画布中心
- 使用 `viewBox="-n -n 2n 2n"` 格式便于居中
- 碰撞盒半径与视觉尺寸匹配
