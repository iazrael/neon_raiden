# Boss数值平衡设计文档

## 设计理念

### 核心原则
1. **血量-攻击力反比平衡**：高血量→低攻击，低血量→高攻击
2. **速度-攻击力平衡**：速度慢→攻击强，速度快→攻击弱
3. **开火频率-伤害平衡**：高频率→低伤害，低频率→高伤害
4. **渐进式难度曲线**：整体线性增长，但根据特质有波动
5. **差异化设计**：每个Boss都有独特的优缺点和战斗风格
6. **最终Boss强势**：第10关全面强化但过渡平滑

## Boss详细配置

### 第1关 - 守护者 (Guardian)
**定位**: 平衡型新手教学Boss

**特点**:
- 标准模板，无明显优缺点
- 作为基准供其他Boss对比

**数值设定**:
- **血量**: 1500（基准值，提升后）
- **速度**: 1.2（中等）
- **弹幕**: 8颗环形，速度4.0
- **开火频率**: 0.06（中等，提升后增强基础难度）
- **移动模式**: 正弦波动
- **武器系统**: 环形弹幕

**设计意图**: 让玩家熟悉基本游戏机制和Boss战斗流程

## 难度曲线分析

### 血量曲线
- 第1-2关：平缓增长（1500→1400）
- 第2-3关：跳跃（1400→2400）
- 第3-5关：稳定增长（2400→2800）
- 第5-7关：快速提升（2800→5000）
- 第7-8关：**反常下降**（5000→4000，爆发刺客特性）
- 第8-9关：恢复增长（4000→6000）
- 第9-10关：大幅跳跃但平滑（6000→10000，1.67倍）

### 速度曲线
- 波动式设计：快慢交替
- 慢速Boss（0.8-1.3）：第3、7关（坦克型）
- 快速Boss（2.0-2.3）：第2、8、10关（刺客/终极型）

### 攻击强度曲线
- 开火频率：0.04（低）→ 0.10（高）
- 弹幕密度：8颗 → 30颗
- 整体线性增长，第5关（24颗）和第8-10关为高峰

---

## 设计亮点

1. **反常设计**: 第8关血量低于第7关，突出爆发刺客定位
2. **平滑过渡**: 第9-10关血量从6000→10000（1.67倍），避免过度突变
3. **差异化**: 每个Boss都有独特战斗风格和明确优缺点
4. **渐进式**: 整体难度线性增长，但有节奏变化
5. **基准提升**: 第1关作为基准，血量和开火频率均有提升
