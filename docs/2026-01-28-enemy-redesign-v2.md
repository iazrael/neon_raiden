# 敌人视觉重新设计文档

**项目：** 霓雷电闪 (Neon Raiden)
**文档日期：** 2026-01-28
**设计目标：** 统一敌人风格为卡通+写实具象造型

---

## 设计原则

### 核心定位
每个敌人都应该是具象的飞机/飞船造型，能一眼识别出是什么，同时保持卡通风格的可爱感。

### 风格参考（保留的优秀设计）

**NORMAL - 侦察机（标准）**
- 红色流线型飞机
- 能量核心：三层圆，透明度渐变
- 造型： recognizable 飞机轮廓

**TANK - 坦克（标准）**
- 绿色坦克
- 履带、炮塔、炮管清晰可见
- 能量核心：三层圆发光

### 统一设计元素

1. **能量核心**：三层圆/菱形，透明度 0.3 → 0.5 → 1.0
2. **配色**：每个敌人独特的主色
3. **造型**：具象的飞机/飞船，不是抽象形状
4. **细节**：炮管、机翼、推进器等可见
5. **尺寸**：大部分 48x48 或 64x64，大单位 80x80
