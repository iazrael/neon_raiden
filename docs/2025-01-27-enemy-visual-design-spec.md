## 设计原则与框架

### 总体设计理念

**核心定位：** 通过视觉语言快速传达敌人的游戏行为模式和威胁等级

**艺术风格：** 增强型赛博朋克几何风格
- **基础：** 简洁的几何形状（圆形、三角形、多边形）
- **增强：** 能量发光效果、装甲纹理、武器外显、动态暗示
- **配色：** 高饱和度霓虹色系 + 深色科技感底色
- **统一元素：** 能量核心（发光点）、推进器光效、武器发射口

### 设计维度映射表

| 配置属性 | 视觉表达要素 |
|---------|------------|
| **移动速度** | 机体流线程度、推进器数量/大小、整体形状的动感 |
| **血量/装甲** | 机体厚度、装甲板数量、护盾光圈、整体尺寸 |
| **行为模式** | 特征性结构（追踪者的眼睛、冲撞者的尖刺、布雷器的挂载点） |
| **武器类型** | 外露炮管、发射舱、能量聚集点、弹仓可视化 |
| **稀有度** | 细节复杂度、发光效果层次、特殊装饰元素 |

### 通用设计元素库

#### 1. 能量核心系统
所有敌人都应有可见的能量核心（圆形/菱形发光点）
- 核心颜色反映威胁等级：白色 < 青色 < 橙色 < 红色 < 紫色
- 核心脉动暗示：使用半透明圆环体现能量流动

#### 2. 推进器系统
- **位置：** 机体后方/下方
- **表现：** 拖尾效果、推进光束
- **大小/数量：** 与速度成正比

#### 3. 武器外显系统
- **直接式：** 炮管伸出机体（FAST, TANK, BARRAGE）
- **内嵌式：** 发射舱门（NORMAL, FORTRESS）
- **能量式：** 无物理武器，只有能量聚集点（KAMIKAZE, PULSAR）

#### 4. 装甲层级
- **轻装甲：** 单层薄壳，透明/半透明
- **中装甲：** 2-3层装甲板，有明显接缝
- **重装甲：** 4+层复合装甲，铆钉/加固点可见

## 普通级敌人 (Common)

### 1. NORMAL - 侦察机 (Scout)

**配置定位：** 基础炮灰 | 血量30 | 速度100 | 正弦波移动 | 普通武器
**稀有度：** Common

#### 视觉设计方案

```
基础形状：倒三角形机体
  - 主体：等边三角形（尖端向下），宽40px，高35px
  - 比例：扁平，体现"侦察"特征

配色方案：
  - 主色：#C05621 橙棕色（廉价、批量生产感）
  - 装甲：#ED8936 浅橙色
  - 能量核心：#FBD38D 淡黄色
  - 武器口：#FC8181 浅红色

结构细节（从内到外）：
  1. 中央圆形能量核心（直径8px，发光）
  2. 三角形主机体（单层薄装甲，厚度3px）
  3. 两侧小型推进器（圆形，直径6px）
  4. 底部单管武器发射口（矩形6x4px）

行为特征可视化：
  - 正弦波移动：机体左右两侧各有1片小型水平翼（15x8px）
  - 暗示机动性：整体形状轻盈，无厚重感

武器外显：
  - 底部中央1个小型炮管（长度5px）
  - 无复杂火控系统（体现基础单位定位）

SVG优化建议：
  - 添加渐变：从核心向外的径向渐变（#FBD38D → #C05621）
  - 推进器：添加半透明拖尾效果（opacity: 0.6）
```

**尺寸：** 48x48px | **碰撞盒半径：** 20px

## 稀有级敌人 (Uncommon)

### 4. KAMIKAZE - 自杀尖刺 (Kamikaze)

**配置定位：** 自杀式冲撞 | 血量5 | 速度200 | 追踪移动 | 无武器
**稀有度：** Uncommon

#### 视觉设计方案

```
基础形状：带刺的能量球
  - 核心：圆形，直径25px
  - 尖刺：8根放射性尖刺（从核心向外）

配色方案：
  - 核心：#FC8181 亮红色（危险警示）
  - 尖刺：#C53030 深红色
  - 能量场：#FEB2B2 粉红色（半透明）
  - 推进器：#E53E3E 鲜红色

结构细节：
  1. 中央圆形能量核心（直径12px，强烈发光）
  2. 环形斥力场（半透明圆环，直径20px，opacity: 0.4）
  3. 8根尖刺呈放射状分布（每根长12px，宽度从3px递减到1px）
  4. 尖刺根部有能量脉动线（橙色细线）

行为特征可视化：
  - 冲撞意图：尖刺全部指向移动方向
  - 高速爆发：后方3个小型推进器（三角形单元，紧凑排列）
  - 无装甲设计：一切为了速度和伤害

武器外显：
  - 无传统武器
  - 整个机体就是"武器"：尖刺本身就是攻击手段
  - 核心超亮发光（暗示"撞上去会爆炸"）

特殊效果：
  - 尖刺有渐变效果（从根部的橙色到尖端的白色）
  - 能量场有脉动动画（半径轻微扩大缩小）
  - 推进器拖尾呈闪电状（不规则路径）

动画建议：
  - 整体旋转：尖刺缓慢旋转（每秒30度）
  - 追踪时加速：靠近玩家时，推进器拖尾明显变长
```

**尺寸：** 48x48px | **碰撞盒半径：** 15px

  - 外环：#805AD5 紫色（能量武器特征）
  - 内核：#D6BCFA 浅紫色
  - 能量流：#E9D8FD 亮紫色（发光）
  - 武器口：#FAF5FF 白紫色

结构细节：
  1. 内层圆形脉冲核心（直径15px，脉动发光）
  2. 4根辐条连接臂（宽度4px，从核心向外辐射）
  3. 外环框架（厚度5px，圆形轨道）
  4. 外环上有4个武器发射口（对称分布）

  - 高射速：4个武器口（每个方向1个）

  - 外环旋转：缓慢旋转（每秒15度，与闪烁同步）

动画建议：
  - 发射时：4个发射口同时闪光
  - 移动时：外环轻微摆动（模拟"脉冲"推进）

### 技术实施建议

#### 1. 使用 SVG 滤镜增强效果
```xml
<defs>
  <!-- 发光效果 -->
  <filter id="glow">
    <feGaussianBlur stdDeviation="2.5" result="coloredBlur"/>
    <feMerge>
      <feMergeNode in="coloredBlur"/>
      <feMergeNode in="SourceGraphic"/>
    </feMerge>
  </filter>

  <!-- 能量流动 -->
  <filter id="energy-flow">
    <feTurbulence type="fractalNoise" baseFrequency="0.05" numOctaves="2"/>
    <feDisplacementMap in="SourceGraphic" scale="3"/>
  </filter>
</defs>
```

#### 2. 保持文件大小优化
- 复用几何元素（`<defs>` + `<use>`）
- 使用简单的渐变而非复杂纹理
- 动画使用 CSS/SVG 原生特性

#### 3. 碰撞盒对齐
- 所有 SVG 的中心点在 (24,24) 或 (40,40)
- 碰撞盒半径与视觉尺寸匹配

#### 4. 代码组织建议

### 与现有精灵图对比

1. **能量核心系统** - 大部分敌人缺少明显的能量核心
